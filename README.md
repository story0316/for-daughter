# 벚꽃길 숨은그림찾기

벚꽃길을 배경으로 화면을 터치해서 숨은 그림을 찾는 모바일 웹 게임입니다. 총 5개 판으로 구성되어 있습니다.

## 실행 방법

빌드 과정 없이 정적 파일만으로 동작합니다. 로컬에서 열어보려면:

```bash
python3 -m http.server 8000
```

브라우저에서 `http://localhost:8000` 접속 후 플레이하면 됩니다. (모바일 화면 크기에서 가장 잘 보입니다.)

## 구성

- `index.html` — 화면 구조 (시작 / 게임 / 클리어 / 완료 화면)
- `style.css` — 스타일
- `script.js` — 배경/숨은 그림 렌더링, 터치 판정, 게임 진행 로직
